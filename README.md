# เกมชักกะเย่อการประมาณค่าทศนิยม

เกมเว็บสำหรับนักเรียนชั้นประถมศึกษาปีที่ 5  
เนื้อหา: การปัดทศนิยม 1 ตำแหน่งเป็นจำนวนเต็มที่ใกล้เคียงที่สุด

## วิธีใช้งาน
1. เปิดไฟล์ `index.html` เพื่อเล่นแบบออฟไลน์
2. หรืออัปโหลดไฟล์ `index.html` ไปยัง GitHub Repository
3. เปิด Settings > Pages
4. เลือก Deploy from a branch
5. เลือก `main` และ `/ (root)`
6. กด Save และรอประมาณ 1–3 นาที

ลิงก์จะมีรูปแบบ:
`https://ชื่อผู้ใช้.github.io/decimal-tug-game/`

## การเล่น
- แบ่งนักเรียนเป็นทีมสีน้ำเงินและทีมสีแดง
- ทั้งสองทีมตอบจากปุ่มของฝั่งตัวเองพร้อมกัน
- ตอบถูกจะได้ 1 คะแนน และดึงริบบิ้นเข้าหาฝั่งตนเอง
- ทีมที่ดึงถึงเส้นชัยก่อน หรือมีคะแนนมากกว่าเมื่อหมดเวลา เป็นผู้ชนะ

พัฒนาเพื่อการเรียนรู้โดย ครูเกต วรรณนิศา  
โรงเรียนบ้านใหม่เสรีธรรม
